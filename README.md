Kanban Board

A task management web app built with vanilla JavaScript and Tailwind CSS, no drag and drop libraries, just native HTML5 APIs.

Features

- Drag and drop tasks between columns (To Do / In Progress / Done) using native HTML5 Drag and Drop API
- Add, update, and delete tasks
- Filter tasks by priority (High / Moderate / Low)
- Search tasks by name
- Data persisted via JSON Server REST API

Tech Stack

- HTML5, CSS3, Tailwind CSS
- Vanilla JavaScript, no frameworks
- JSON Server REST API

Run Locally

1. Clone the repo: git clone https://github.com/lenahmahmoud/KanbanBoard.git
2. Install dependencies: npm install
3. Start JSON Server: npx json-server --watch db.json --port 3000
4. Open index.html in your browser
